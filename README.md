## ✨ Features

- Fully **responsive** — works on mobile, tablet and desktop
- **Smooth scroll** animations and reveal effects on scroll
- **Typing animation** in hero section
- **Skill progress bars** for all technologies
- **Project cards** with direct GitHub links
- **Download CV** button
- **Contact section** with email, LinkedIn and GitHub links
- Clean **dark theme** with colorful purple, pink and cyan accents
